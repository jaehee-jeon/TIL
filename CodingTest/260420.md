## 문제 1 - 정수 찾기
```python
def solution(num_list, n):
    answer = 0
    if n in num_list :
        answer = 1
    
    return answer
```
- `if` 연산자로 리스트 포함 여부 확인 가능

## 문제 2 - 문자열 곱하기
```python
def solution(my_string, k):
    answer = ''
    
    answer = my_string*k
    return answer
```
- 문자열 곱하기 가능

## 문제 3 - 짝수 홀수 판별
```python
def solution(arr, delete_list):
    answer = []
    
    answer = [i for i in arr if i not in delete_list ]
    return answer
```
- `리스트 컴플레이션` [ "넣을 값" | "어디서 꺼내나" | "조건이 뭔가" ]
- `not in` 포함되어 있지 않은 경우
